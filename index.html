Pipeline de Dados: Cinema Nacional

Engenharia de Dados & Business Intelligence

Este projeto realiza o ciclo completo de Engenharia de Dados: desde a ingestão automatizada de dados de arrecadação, salas e catálogos do portal público do governo federal (GovBr), passando pela limpeza e modelagem dimensional em banco de dados, até a entrega de dashboards interativos e painéis analíticos.

Tratamentos de Dados & Pipeline:

  • Ingestão Automática: Requisições HTTP robustas direto do portal GovBr para obtenção automatizada dos arquivos brutos compactados de arrecadação, salas e catálogos.
  • Descompactação & Sanitização: Script Python que descompacta os arquivos, limpa caracteres especiais de strings, trata valores nulos e realiza a tipagem correta de campos numéricos e monetários.
  • Armazenamento & dbt: Carga dos dados tratados no PostgreSQL e modelagem estruturada em camadas Bronze (raw), Prata (limpos e deduplicados) e Gold (dimensões e fatos otimizadas para BI).
  • Orquestração: Apache Airflow orquestrando tarefas dentro de containers Docker, gerenciando retentativas, logs de erros e controle de fluxo diário.

Pipeline de Dados: Eleições Municipais

Engenharia de Dados & Business Intelligence

Este projeto realiza o ciclo completo de Engenharia de Dados voltado para dados eleitorais: desde a extração automatizada de perfis de eleitores, locais de votação e boletins de urnas diretamente dos portais abertos do TSE, passando pela limpeza e modelagem dimensional em banco de dados, até a visualização analítica interativa de dados demográficos e de apuração.

Tratamentos de Dados & Pipeline:

  • Ingestão Automática: Download automatizado de grandes volumes de dados brutos de urnas, perfil de eleitores e locais de votação do portal público do TSE.
  • Descompactação & Sanitização: Script Python que gerencia a descompactação de arquivos brutos ZIP, realiza limpeza de caracteres especiais, trata nulos e define a tipagem correta dos dados.
  • Armazenamento & Modelagem (dbt): Carga no PostgreSQL e estruturação em camadas Bronze, Prata e Gold, consolidando dados e aplicando filtros por Ano e Estado para alta performance em BI.
  • Orquestração & Governança: Fluxo completo orquestrado pelo Apache Airflow rodando em containers Docker, com rotinas de arquivamento para manter o ambiente de processamento limpo e auditável.

Pipeline de Dados: Dinâmica Empresarial (CNPJ)

Engenharia de Dados & Business Intelligence

Este projeto implementa um pipeline híbrido e escalável para processar os dados da base pública de CNPJs da Receita Federal (GovBr), permitindo análises estratégicas de abertura, fechamento e saúde das empresas no município.

Tratamentos de Dados & Pipeline:

  • Ingestão Escalável: Download automatizado de arquivos brutos ZIP compactados diretamente da Receita Federal (GovBr) via requisições HTTP, com upload imediato para buckets do Google Cloud Storage (GCS).
  • Processamento Híbrido: Tratamento leve (limpeza, tipagem e correção de nulos) utilizando scripts Python (Pandas) para dimensões e tabelas menores, e processamento pesado distribuído usando Spark no Kubernetes para a tabela massiva de Estabelecimentos.
  • Armazenamento & dbt: Carga dos dados tratados no Google BigQuery e modelagem dimensional em dbt através de uma arquitetura medalhão (Bronze, Silver e Gold), facilitando a criação de visões fiscais e auditorias.
  • Orquestração & Governança: Fluxo completo orquestrado pelo Apache Airflow com controle rigoroso de concorrência, retentativas automáticas em caso de falhas da API federal e eliminação automática de arquivos temporários.

Pipeline de Dados: Desempenho Comercial (Loja)

Engenharia de Dados & Business Intelligence

Este projeto demonstra o ciclo completo de Business Intelligence e Engenharia de Dados focado no varejo comercial. O pipeline inicia-se com a extração e restauração de backups gerados por um sistema ERP de vendas, prossegue com tratamentos avançados via queries SQL no PostgreSQL para consolidação e estruturação dimensional do banco de dados, e finaliza com a criação de painéis gerenciais altamente analíticos no Power BI.

Tratamentos de Dados & Pipeline:

  • Restauração & Carga: Importação direta a partir de backups de banco de dados gerados por um ERP comercial real, simulando perfeitamente a dinâmica operacional de uma loja.
  • ETL via SQL no PostgreSQL: Processo de tratamento de dados centralizado no banco de dados através de queries SQL robustas para sanitização, correção de nulos e modelagem relacional.
  • Modelagem Dimensional: Estruturação e relacionamento de tabelas de dimensão fundamentais (Produtos, Vendedores, Clientes, Parceiros, Estoque) com a tabela fato de Vendas, facilitando as análises de cruzamento de dados.
  • Dashboards no Power BI: Entrega de visões estratégicas contendo indicadores de faturamento, rentabilidade, desempenho da equipe de vendas, comportamento dos clientes e movimentações de estoque.

Gestão de Contratos: Projeto Tapajós

Business Intelligence & Análise Geográfica

Este projeto consiste na estruturação, tratamento e consolidação de dados contratuais. A partir de dados brutos disponibilizados em tabelas Excel desestruturadas, foi realizado um processo completo de ETL para padronização das informações de contratos. O diferencial analítico do projeto inclui a plotagem geográfica de pontos de localização exatos via ArcGIS e a criação de mapas temáticos interativos utilizando SVG, permitindo uma análise espacial avançada e intuitiva da abrangência dos contratos.

Tratamentos de Dados & Diferenciais:

  • Saneamento & ETL: Consolidação de planilhas Excel desestruturadas, aplicando regras de negócio para tratamento de valores nulos, tipagem de dados (valores e datas) e remoção de duplicidades.
  • Pontos de Localização (ArcGIS): Integração com ArcGIS para plotagem geoespacial precisa dos pontos onde os contratos estão ativos ou sendo executados.
  • Visualização Customizada em SVG: Implementação de mapas SVG customizados e interativos no painel para facilitar a navegação territorial e destacar indicadores por região.
  • Painel Gerencial no Power BI: Consolidação de todas as métricas em um painel interativo, correlacionando indicadores financeiros e operacionais com dados geográficos.

Pipeline de Dados: Monitoramento Orçamentário

Engenharia de Dados & Business Intelligence

Este projeto consiste no desenvolvimento de uma solução analítica completa para o monitoramento e fiscalização da execução orçamentária municipal. A partir de dados brutos extraídos diretamente da base do sistema de gestão do município, foi estruturado um Data Warehouse em SQL Server. O pipeline de ETL realiza a leitura, cruzamento e consolidação de tabelas de fontes de recursos, receitas, despesas e unidades gestoras, alimentando um dashboard interativo focado na detecção de desvios e controle de gastos.

Tratamentos de Dados & Pipeline:

  • Extração de Origem: Ingestão dos lançamentos de receitas e despesas diretamente da base de dados do sistema de contabilidade e gestão pública municipal.
  • ETL & Unificação: Cruzamento sistemático de tabelas de fontes de recursos, unidades gestoras, receitas orçadas e despesas empenhadas/liquidadas.
  • Data Warehouse (SQL Server): Modelagem dimensional e armazenamento estruturado para garantir integridade analítica e alta performance nas consultas de BI.
  • Indicadores Estratégicos: Painel gerencial focado em evidenciar a utilização dos recursos, identificando variações entre valores planejados e executados e analisando gastos por unidade executora.
